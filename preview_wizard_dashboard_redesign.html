Layer 3 · Nav clarity (smaller fix)

Wizard URL /wizard/710 renders step 2 (Foundation), but the page <h1> still says "Select Entity" (step 1). Also no breadcrumb back to dashboard.

Current

<h1>Select Entity</h1>     ← always says this
<div>STEP 2 OF 5</div>     ← step number visible in body
<h2>Foundation</h2>       ← actual step name buried

Browser tab + back button context = "Select Entity" forever. On refresh you can't tell which step you were on without scanning.

Proposed recommended

<h1>Foundation — 157 East 72nd (710)</h1>
<div class="crumb">
  <a href="/dashboard">← FA Dashboard</a>
  · Step 2 of 5
</div>

Tab title reflects current step + entity. One-click breadcrumb to dashboard. Refresh restores the right header. Step number stays visible.
