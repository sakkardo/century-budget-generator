What changed in v2

  • Invoice drill is back, untouched. Click GL code or Description → row expands with the same invoice table PMs use today (Payee, Description, Inv#, Date, Amount, Check#, Reclass action). Same muscle memory.
  • Reference numbers ride along with the invoice drill. When the row expands, the top of the detail panel shows 6 reference tiles (Prior Year, YTD, Estimate, 12mo Forecast, $ Variance, % Change). Then the invoice table below. One click reveals everything the PM needs to decide.
  • Main table compacts from 14 columns to 7. Identity → Curr Budget → Proposed → Inc $ → Inc % → Notes. Less scanning, more focus.
  • Single color budget per row: 3 treatments. Gray text for reference, blue input for action, red/green pills for delta. fx stickers gone. Yellow notes border gone (replaced with a tiny red dot when notes are required).
  • Inline "= " icon button replaces the "Mark No Change" button under the input. Same row height every row.
  • Subtotal rows render derived pills. No more visual gaps. Per-category rollup delta visible at a glance.
