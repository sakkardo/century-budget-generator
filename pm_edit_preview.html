📐 Either-or logic — how the two new columns interact

Scenario A: PM enters Increase % = 5%

→ Proposed Budget = current_budget × (1 + 5%)
→ Increase $ column auto-clears (blank).
→ Database stores: increase_pct=0.05, increase_dollar=NULL

Scenario B: PM enters Increase $ = $1,500

→ Proposed Budget = current_budget + $1,500
→ Increase % column auto-clears (blank).
→ Database stores: increase_pct=NULL, increase_dollar=1500

Scenario C: PM clears both

→ Proposed Budget = current_budget (no change).
→ Database stores: increase_pct=NULL, increase_dollar=NULL
